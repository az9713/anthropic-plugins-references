For Git & Development Workflow

commit-commands

Never write a lazy git commit message again
  • /commit — auto-generates meaningful messages
  • /commit-push-pr — full workflow in one shot
  • /clean_gone — cleans up stale branches
  • Infers commit style from your git history
Every developer — stops "fix stuff" commit messages forever.

feature-dev

7-step structured process from idea to done
  • Discovery → exploration → architecture → implementation → review
  • 3 specialized agents (explorer, architect, reviewer)
  • Stops you skipping important steps
Side-project builders who want a disciplined process.

ralph-loop

Claude iterates on a task until it's done
  • Runs Claude in a loop until completion criteria met
  • --max-iterations safety limit
  • Preserves file changes & git history
Automating repetitive coding tasks — set it and come back.

hookify

Custom automation triggers — no coding required
  • Define triggers using markdown config files
  • Pattern-based (Python regex matching)
  • Actions: warn, block, or run something
  • /hookify, /hookify:list, /hookify:configure
Power users who want Claude to behave differently in certain situations.
